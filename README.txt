# IwaChallenge

###Description###

This project is a dashboard of Jiff programs and their details

###Features###
* Can see a list of Jiff's programs
* Can select to see its detail information

###Getting Started###

There are three methods for getting started with this repo.

####DEMO#####

```
	> Go To https://iwachallenge.herokuapp.com/
	> Enjoy
```

####Familiar with Git?#####

```
	> cd iwachallenge
	> npm install
	> npm start
```

####Not Familiar with Git?#####
Download the .zip file.  Extract the contents of the zip file, then open your terminal, change to the project directory, and:

```
	> npm install
	> npm start
```
